# Role: 시니어 일렉트론(Electron) 및 프론트엔드 개발자

# Project: '인생네컷' 스타일의 데스크톱 포토부스 어플리케이션 제작

# Tech Stack: Electron, React (Vite), Tailwind CSS, Canvas API, Lucide React (아이콘)

# Core Logic:

1. **카메라 연동**: `navigator.mediaDevices.getUserMedia`를 사용하여 실시간 웹캠 미리보기를 구현해줘. (사용자 편의를 위해 좌우 반전 필터 적용)
2. **촬영 시퀀스**:
   - '촬영 시작' 버튼 클릭 시 8초 간격으로 총 8번 자동 촬영.
   - 각 촬영 전 3초 전부터 화면에 카운트다운 숫자 표시.
   - 촬영 시 셔터 효과(화면 깜빡임)와 소리 추가.
3. **사진 선택 및 합성**:
   - 촬영된 8장의 썸네일을 보여주고 사용자가 4장을 순서대로 선택하도록 함.
   - 선택된 4장을 Canvas API를 사용하여 '세로형 4컷 레이아웃(프레임)'으로 합성.
   - 프레임 하단에 날짜나 로고가 들어갈 공간 확보.
4. **공유 및 저장**:
   - 합성된 이미지를 로컬에 저장하는 기능.
   - (중요) '공유하기' 버튼 클릭 시, 이미지를 임시 서버(예: Supabase Storage)에 업로드하고 해당 URL을 QR 코드로 변환하여 화면에 띄워주는 로직 제안.

# Task:

- 위 기능을 구현하기 위한 전체적인 `App.jsx` 또는 컴포넌트 구조 코드를 작성해줘.
- 특히 8초 간격의 타이머 로직과 Canvas를 이용한 이미지 합성 함수(`handleCombineImages`)를 중점적으로 짜줘.
- 깔끔하고 모던한 포토부스 키오스크 스타일의 UI 코드를 Tailwind CSS로 작성해줘.
